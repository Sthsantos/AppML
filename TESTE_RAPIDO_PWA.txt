🧪 COMO TESTAR AGORA - PASSO A PASSO
================================================

OPÇÃO 1: TESTE NO NAVEGADOR DO COMPUTADOR
------------------------------------------

1. Abra o navegador Chrome ou Edge

2. Acesse: http://127.0.0.1:5000

3. Pressione F12 (DevTools)

4. Vá para a aba "Console"

5. Aguarde 3 segundos e observe:
   - Deve aparecer: "✅ Service Worker registrado com sucesso"
   - Banner de instalação aparece na tela

6. Se não aparecer, cole no Console:
   document.getElementById('pwaInstallPrompt').style.display = 'block';

7. Clique em "Instalar Agora"


OPÇÃO 2: TESTE NO CELULAR (RECOMENDADO)
----------------------------------------

1. No computador, descubra seu IP:
   - Abra Prompt de Comando
   - Digite: ipconfig
   - Anote o "Endereço IPv4" (ex: 192.168.1.100)

2. No celular (mesma rede Wi-Fi):
   - Abra Chrome
   - Digite: http://SEU_IP:5000
   - Exemplo: http://192.168.1.100:5000

3. Aguarde 3 segundos → Banner aparece!

4. Toque em "Instalar Agora"

5. Confirme → App instalado! 🎉


OPÇÃO 3: VERIFICAÇÃO MANUAL DAS ROTAS
--------------------------------------

Abra no navegador:

1. http://127.0.0.1:5000/sw.js
   ✅ Esperado: Código JavaScript (Service Worker)

2. http://127.0.0.1:5000/manifest.json
   ✅ Esperado: JSON com dados do app

3. http://127.0.0.1:5000
   ✅ Esperado: Página inicial carrega normalmente


OPÇÃO 4: USAR O BOTÃO DO MENU
------------------------------

1. Acesse: http://127.0.0.1:5000

2. Faça login (se necessário)

3. Clique no menu lateral (☰)

4. Procure o botão "Instalar App" 
   (aparece só se instalação disponível)

5. Clique no botão → Instala!


================================================
🔍 SE NÃO APARECER O BANNER
================================================

MOTIVOS COMUNS:

1. App já instalado
   → Verifique ícone na tela inicial

2. Banner foi dispensado antes
   → Aguarde 7 dias ou limpe:
   - F12 → Console
   - Digite: localStorage.clear();
   - Recarregue a página

3. Navegador não suporta
   → Use Chrome ou Edge

4. Modo anônimo/privado
   → Use modo normal

5. Service Worker não registrou
   → Veja Console (F12) se há erros

localStorage.clear();
navigator.serviceWorker.getRegistrations()
    .then(regs => regs.forEach(r => r.unregister()));
location.reload();

# Verificar se está instalado
console.log('Instalado:', 
    window.matchMedia('(display-mode: standalone)').matches);
